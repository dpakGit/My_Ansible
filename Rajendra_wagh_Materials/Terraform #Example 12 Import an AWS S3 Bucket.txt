Example: Import an AWS S3 Bucket
Step 1 ‚Äî Create the Terraform resource block

Create a file main.tf:

provider "aws" {
  region = "us-east-1"
}

resource "aws_s3_bucket" "demo" {
  # Intentionally empty ‚Äî Terraform will fill details after import
}


‚ö†Ô∏è Do NOT write bucket = "name" yet, because Terraform doesn't know it until import.

Step 2 ‚Äî Run the import command

Use your existing bucket name:

terraform init
terraform import aws_s3_bucket.demo my-existing-bucket-name


This will output something like:

aws_s3_bucket.demo: Import prepared!
  Prepared aws_s3_bucket for import
aws_s3_bucket.demo: Import complete!
  Imported aws_s3_bucket (ID: my-existing-bucket-name)

Step 3 ‚Äî See the imported data

Run:

terraform show


Terraform will display everything it discovered about the S3 bucket.

Step 4 ‚Äî Update your resource block

After import, you must update the resource block so it matches AWS.

Example:

resource "aws_s3_bucket" "demo" {
  bucket = "my-existing-bucket-name"
}
