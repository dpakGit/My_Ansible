### Ansible demo

**Question**

Create an Ansible playbook that:

- Installs Apache HTTP Server on all hosts
- Starts the Apache service
- Removes any existing files in the /var/www/html/ directory
- Installs Git on the hosts
- Clones a Git repository (ansibledemo) from GitHub to the /var/www/html/ directory, which contains an index.html file

OR

**Question**

Write an Ansible playbook that sets up a web server on all hosts with the following configuration:

- Installs Apache HTTP Server
- Starts the Apache service
- Deploys an index.html file from a GitHub repository (ansibledemo) to the /var/www/html/ directory
